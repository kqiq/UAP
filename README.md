### Prerequisites
- Node.js 18+

### Quick Setup

1. **Clone the repository**
```bash
git clone https://github.com/kqiq/uap.git
cd uap
```

2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```

4. **Access Admin Panel**
Navigate to `http://localhost:5173` and login with admin credentials
